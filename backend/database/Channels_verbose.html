CHANNEL ( id_channel, name_channel )
  • Le champ id_channel constitue la clé primaire de la table. C'était déjà un identifiant de l'entité CHANNEL.
  • Le champ name_channel était déjà un simple attribut de l'entité CHANNEL.
IS_IN_CHANNEL ( #nickname_user, #id_channel, is_admin )
  • Le champ nickname_user fait partie de la clé primaire de la table. C'est une clé étrangère qui a migré directement à partir de l'entité USER.
  • Le champ id_channel fait partie de la clé primaire de la table. C'est une clé étrangère qui a migré directement à partir de l'entité CHANNEL.
  • Le champ is_admin était déjà un simple attribut de l'association IS_IN_CHANNEL.
MESSAGE ( id_message, time_message, #id_channel, #nickname_user )
  • Le champ id_message constitue la clé primaire de la table. C'était déjà un identifiant de l'entité MESSAGE.
  • Le champ time_message était déjà un simple attribut de l'entité MESSAGE.
  • Le champ id_channel est une clé étrangère. Il a migré par l'association de dépendance fonctionnelle BELONGS_TO à partir de l'entité CHANNEL en perdant son caractère identifiant.
  • Le champ nickname_user est une clé étrangère. Il a migré par l'association de dépendance fonctionnelle BELONGS_TO à partir de l'entité USER en perdant son caractère identifiant.
